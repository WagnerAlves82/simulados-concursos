# Simulados de Concursos

Este projeto permite criar e gerenciar simulados de concursos online.

## Funcionalidades
- Visualizar questões por área de conhecimento.
- Cadastrar e gerenciar simulados.
- Interface moderna em React/Next.js.

## Tecnologias
- Next.js
- React
- Node.js
- Tailwind CSS
